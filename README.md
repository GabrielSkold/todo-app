# Todo App

A full-stack todo app built with:

- Node.js
- Express
- MongoDB
- Vanilla JavaScript

## Features

- Add todos
- Delete todos
- Mark todos as complete
- Edit todos

## How to run locally

1. Clone the repo
2. Install dependencies:

   npm install

3. Create a .env file:

   MONGO_URI=your_mongo_uri
   PORT=3000
   DB_NAME=ToDo

4. Start the server:

   npm start

5. Open in browser:

   http://localhost:3000
